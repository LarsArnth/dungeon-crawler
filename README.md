# Game Hub

A browser-based game hub featuring multiple mini-games, hosted on GitHub Pages.

## Games

- **Dungeon Crawler** - Roguelike dungeon crawler with procedural generation, combat, inventory, and save/load
- **Snake** - Classic Snake game
- **Space Invaders** - Space Invaders clone

## Structure

```
index.html        # Main menu hub with game cards
games/
  dungeon-crawler.html
  snake.html
  space-invaders.html
```

## Deployment

Hosted on GitHub Pages: https://github.com/LarsArnth/dungeon-crawler

## Development

When making changes to game code:
1. Edit files in the repository
2. Commit with descriptive messages: `git add -A && git commit -m "message"`
3. Push to main: `git push origin main`
4. Enable GitHub Pages in repo settings (Settings > Pages > Source: main branch)
